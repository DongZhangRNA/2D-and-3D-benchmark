## This repository stores the 2D and 3D structure predictions of R1108 from [CASP15](https://predictioncenter.org/casp15/index.cgi)

### The corresponding RNA 2D structure prediction tools for the files (.dbn) in the `Predictions_2D` folder are listed as follows:

`RNAfold.dbn`: [RNAfold v2.6.4](http://www.tbi.univie.ac.at/RNA/),\
`RNAStructure.dbn`: [RNAStructure v6.4](https://rna.urmc.rochester.edu/RNAstructure.html),\
`contrafold.dbn`: [CONTRAfold v2.02](http://contra.stanford.edu/contrafold/),\
`mfold.dbn`: [Mfold v3.6](https://www.unafold.org/mfold/software/download-mfold.php),\
`nupack.dbn`: [NUPACK v4.0](https://nupack.org/download/overview),\
`mxfold2.dbn`: [MXfold2](https://github.com/mxfold/mxfold2), \
`spot.dbn`: [SPOT-RNA](https://github.com/jaswindersingh2/SPOT-RNA),\
`UFold.dbn` (webserver): [UFold](http://146.56.237.198:3838/UFold/), \
`CentroidFold_McCaskill_BL.dbn` and `CentroidFold_McCaskill_Turner.dbn` (webserver): [CentroidFold v0.0.16](http://rtools.cbrc.jp/centroidfold/), \
`PETfold1.dbn` and `Pfold1.dbn`: [PETfold v1.0](https://rth.dk/resources/petfold/download.php),\
`PETfold2.dbn` and `Pfold2.dbn`: [PETfold v2.2](https://rth.dk/resources/petfold/download.php),\
`BPfold.dbn`: [BPfold](https://github.com/heqin-zhu/BPfold),\
`LinearFold.dbn`: [LinearFold](https://github.com/LinearFold/LinearFold), and\
`EternaFold.dbn` and `EternalFoldPoent.dbn`: [EternaFold v.1.3.1](https://github.com/eternagame/EternaFold).

### The corresponding RNA 3D structure prediction tools in the `Predictions_3D` folder are listed as follows:

`trRosettaRNA`: [trRosettaRNA v1.1](https://yanglab.qd.sdu.edu.cn/trRosettaRNA/download/),\
`RNAComposer` (web server): [RNAComposer](https://rnacomposer.cs.put.poznan.pl/),\
`FARFAR2` (`rna_denovo` in `Rosetta` package): [Rosetta](https://www.rosettacommons.org/software/license-and-download),\
`IsRNA2`: [IsRNA2](https://10.1021/acs.jpcb.1c07288), and\
`SimRNA`: [SimRNA v3.20](https://genesilico.pl/software/stand-alone/simrna).
